С реакциями в телеграмм каналах ничего не выйдет, api не предоставляет доступка к реакциям на постам, максимум можно спарсить какие реакции поставлены и количество
Парсинг комментариев со всех постов и получение оттуда юзеров - возможно, но если тгк создан давно и в нем много постов, можно упереться в блокировку от api так как будет много запросов => либо ставить задержку перед парсингом каомментов на каждом посте, либо отказываться от этйо функции. Даже если этой задержки не будет, скрипт будет отрабатывать очень долго.
Парсить подписчков телеграмм канала невозможно, если он не твой или ты в нем не админ.
